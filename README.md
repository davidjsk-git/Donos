# 📊 통합 후원금 및 지출 분석 시스템

이 애플리케이션은 **HOPE**와 **GLFocus** 등 여러 단체의 후원금 내역을 독립적으로 관리하고, 통합 분석 리포트를 통해 후원자 패턴을 심층 분석할 수 있는 데스크톱 대시보드입니다.

## 🚀 실행 방법 (Mac / Linux)

1. **터미널** 앱을 실행합니다.
2. 프로젝트 폴더로 이동합니다: `cd ~/Documents/dev/donationDashboard`
3. 실행 스크립트를 작동시킵니다: `./run_dashboard.sh`

---

## 🚀 실행 방법 (Windows)

1. **파이썬 설치:** [python.org](https://www.python.org/)에서 파이썬(3.10 이상)을 설치합니다. (설치 시 **"Add Python to PATH"**를 반드시 체크하세요.)
2. **프로젝트 복사:** 현재 폴더 전체를 윈도우 컴퓨터로 복사합니다.
3. **실행:** 폴더 내의 `run_dashboard.bat` 파일을 더블 클릭하여 실행합니다.

---

## 💡 주요 기능
- **멀티 단체 관리:** 사이드바에서 HOPE, GLFocus 등 관리할 단체를 선택하여 개별적으로 관리할 수 있습니다.
- **통합 분석 리포트:** 두 단체의 데이터를 합산하여 전체 재무 현황 및 후원자(TOP 5, 연속, 일회성, 불연속) 패턴을 분석합니다.
- **파일 업로드 지원:** 기존 HTML 형식뿐만 아니라 표준 CSV 형식의 파일 업로드도 지원합니다.
- **기초 잔액 설정:** 단체별로 기초 잔액을 설정하여 현재 잔액을 정확하게 산출합니다.

## 💡 주요 기능
- **HTML 업로드:** 제공된 형식의 후원 내역 HTML 파일을 사이드바에서 업로드하여 데이터를 추가할 수 있습니다.
- **내역 직접 편집:** 사이드바의 '내역 직접 추가' 폼을 사용하거나, 메인 화면의 '전체 내역' 테이블에서 직접 데이터를 수정하고 삭제할 수 있습니다.
- **증감 분석:** 전월 대비 후원금 변화액과 신규 후원자/이탈 후원자 목록을 자동으로 분석하여 보여줍니다.
- **시각화:** 월별 수입 및 지출 추이를 그래프로 한눈에 파악할 수 있습니다.
